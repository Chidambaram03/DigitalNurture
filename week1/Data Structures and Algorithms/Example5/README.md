# Task Management System

## Types of Linked Lists

### Singly Linked List
Each node contains data and a reference to the next node.

### Doubly Linked List
Each node contains data, next reference, and previous reference.

## Operations Implemented

- Add Task
- Search Task
- Traverse Tasks
- Delete Task

## Time Complexity

| Operation | Complexity |
|------------|------------|
| Add | O(n) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Advantages of Linked Lists

- Dynamic size
- Efficient insertions and deletions
- No need for contiguous memory allocation

## Linked Lists vs Arrays

Linked Lists are preferred when:
- Data size changes frequently
- Frequent insertions/deletions are required

Arrays are preferred when:
- Fast random access is needed
- Data size is fixed
