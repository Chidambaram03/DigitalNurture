# Library Management System

## Linear Search

Linear Search checks each element one by one.

Time Complexity:
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

## Binary Search

Binary Search works only on sorted data.

Time Complexity:
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Comparison

Linear Search:
- Works on unsorted data
- Simpler implementation
- Slower for large datasets

Binary Search:
- Requires sorted data
- Much faster for large datasets

## Conclusion

Binary Search is preferred for large sorted datasets because it significantly reduces search time.
